# Add a new Dish :

1. Fork the project
2. Clone it 
3. Create new branch "git checkout -b <Your_Dish_Name>
4. Find the index.js  ( within /constants folder)
5. Add a new item into the items array
6. Commit your changes
7. Create a Pull Request

- Make sure the id is unique
